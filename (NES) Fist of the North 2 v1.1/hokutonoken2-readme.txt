-------------------------------------------------
-Hokuto no Ken 2: Seikimatsu Kyuuseishu Densetsu-
-------------------------------------------------

Fist of the North Star 2 is the second NES game released with the Fist of the North Star license.  It was also the only Fist of the North Star game released outside of Japan on the Nintendo Entertainment System.

Note:  This patch is to be made available on GitHub and on RHDI only.  I do not give permission for RHDN to distribute this patch onto their site.

The patch comes in four flavors...
Fist of the North Star 2 (English Re-Translation 1.1).ips
--English translation patch for Japanese ROM (IPS Format)
Fist of the North Star 2 (English Re-Translation 1.1).bps
--English translation patch for Japanese ROM (BPS Format)
Fist of the North Star 2 B (English Re-Translation 1.1).ips
--English translation patch for USA ROM (IPS Format)
Fist of the North Star 2 B (English Re-Translation 1.1).bps
--English translation patch for USA ROM (BPS Format)

Use either the IPS or BPS patch labelled "Fist of the North Star 2 (English Re-Translation 1.1)" on the Japanese NES ROM with the following hashes. (Don't use both patches.)

File SHA-1: 0BA41EC735C5CC1DEA8003335AB899D40038B203
File CRC32: 7539D3D2
ROM SHA-1: F8D8B86A80BA2E1AB6B95E587498CCC08549290D
ROM CRC32: D1F7DF3A

Use either the IPS or BPS patch labelled "Fist of the North Star 2 B (English Re-Translation 1.1)" on the USA NES ROM with the following hashes. (Don't use both patches.)

File SHA-1: C3051CC517D661C164CEDB4FC15D2958AB862CCC
File CRC32: A219206B
ROM SHA-1: F9809D020BC0AFF976CD657E3F923D81EBD6C6B1
ROM CRC32: 6D72C83

The end result should be an NES ROM with the following hashes.

File SHA-1: 14138DA29D0E03B0586A4548D0C9FDE66387465F
